# Easy Option Learning

Options trading education platform.

## Run locally

Requirements:
- Node.js 18+
- npm

```sh
# from project root
cd frontend
npm install
npm run dev
```

## Frontend stack

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui

## Branding and link preview

Share preview metadata is configured in `frontend/index.html`:
- `og:title`
- `og:description`
- `og:image`
- `twitter:image`
